# SF Symbols

### 🚀 Icon Engine (`sf-sym`)
The **Icon Engine** is a lightweight Web Component that renders 6,000+ Apple symbols.

**CDN**
```html
<script src="https://cdn.jsdelivr.net/gh/username/repo@main/script.js"></script>
```

**Usage:**

SF Symbols behave like a standard text character.

It supports CSS styling and the "style" attribute.
```html
<sf-sym name="ant-fill"></sf-sym>
<sf-sym name="gear" style="color: #0a84ff; font-size: 32px;"></sf-sym>
```

**Notes:**

- A list of every SF Symbol is in `icons.txt`.

### 📚 Typography
Use Apple's signature SF Pro font without annoying downloads or applications.

**CDN**
```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/username/repo@main/fonts.css">
```

**Usage:**
```css
h1 {
    font-family: "SF Pro Display", sans-serif;
    font-weight: 800;
}

p {
    font-family: "SF Pro Text", sans-serif;
    font-weight: 400;
}

button {
    font-family: "SF Pro Rounded", sans-serif;
    font-weight: 600;
}
```
